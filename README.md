# ocean-api-access
Code to automatically access the OCEAN API.

To run the code, you need to set the following values in your [global environment variables](https://www3.ntu.edu.sg/home/ehchua/programming/howto/Environment_Variables.html#:~:text=To%20set%20an%20environment%20variable,set%20varname%20%3Dvalue%20%22).).

`OCEAN_USERNAME` (your full email address, including @st-andrews.ac.uk).

`OCEAN_PASSWORD` (your API password, provided by your administrator.)
